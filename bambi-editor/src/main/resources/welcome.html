Welcome to Bambi Editor!

Bambi Editor is an easy to use desktop image authoring tool capable of fast and quick filter transformations and no-nonse printing. Bambi is also a web authoring tool for web developers which allows their end users to upload images scaled exactly to specific web requirements.

BambiEditor is free software, released under GPL. This, as all 0.9x versions is a BETA as we march towards a feature complete 1.0.
